## Description

  API de alunos e atividade desenvolvida em [Nest](https://github.com/nestjs/nest)

## Instalação

```bash
$ npm install
```

## Rodar API

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
